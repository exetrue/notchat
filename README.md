# Notdefteri Güvenli

PIN korumalı Notlar, Hatırlatmalar, Şifrelerim ve Ayarlar uygulaması.

## GitHub üzerinden APK alma
1. Bu klasörü GitHub reposuna yükleyin.
2. GitHub > Actions > Build Android APK iş akışını çalıştırın.
3. İş bitince Artifacts bölümünden `notdefteri-apk` dosyasını indirin.

Yerelde APK almak için:
```bash
flutter pub get
flutter build apk --release
```
